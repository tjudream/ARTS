# Algorithm [890. Find and Replace Pattern](https://leetcode.com/problems/find-and-replace-pattern/)
## 1. 问题描述
给一个单词列表，给一个模式，找出符合模式的单词。

返回符合模式的所有单词列表。

示例：
* 输入: words = ["abc","deq","mee","aqq","dkd","ccc"], pattern = "abb"
* 输出: ["mee","aqq"]
* 解释: "mee" {a -> m, b -> e}, "aqq" {a -> a, b -> q}
